Get Started

Welcome to Grommet get started page. Here you will find instructions on how to install Grommet in your local environment.

Get Dependencies

  1. Install Git. For Windows, you may like Git for Windows.
  2. Install Node.js (at least Node 0.9+ and NPM 1.4.x+ required)

  3. Setup your NPM proxy (only required if you're behind a proxy server).

    
          {'npm config set http-proxy http://\{host\}:\{port\}'}
    
          {'npm config set https-proxy https://\{host\}:\{port\}'}

Get Going

You can install Grommet using either of the methods below.

NPM

  1. Use the Grommet NPM private registry (temporary step while Grommet is not available in public NPM)

    
                {'npm set registry http://grommet.usa.hp.com:2374'}

  2. Install Gulp globally

    
                {'npm install -g gulp'}

  3. Install Grommet globally

    
                {'npm install -g grommet'}

  4. Use the Grommet generator to bootstrap your new app

    
                {'grommet init sample-app'}
    
                {'cd sample-app'}

  5. Start a development server with hot reload enabled

    
                {'gulp dev'}

  6. Open http://localhost:9000/webpack-dev-server/

    At this point you should be able to see the application dashboard running.

  7. Checkout the Modular Grommet page to understand the application structure better.

Bower

  1. Install Bower (at least 1.x required)

    
              {'npm install -g bower'}

  2. Create environment variable with your proxy settings (only required if you're behind a proxy server)

    Windows

    
                {'set HTTPS_PROXY=https://\{host\}:\{port\}'}

    Linux/OS X

    
                {'export HTTPS_PROXY=https://\{host\}:\{port\}'}

  3. Install Grommet

    
                {'bower install --save HewlettPackard/grommet-bower'}

    At this point, a bower_components folder will be created which contains Grommet and its dependencies.

  4. Open sample application in the browser

    bower_components/grommet/sample-grommet.html

    You can now play with Grommet using local Bower.

  5. Checkout the Tutorial page to understand this sample app better.
