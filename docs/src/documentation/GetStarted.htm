Get Started

This section aims to provide you with assets to quickly get Grommet framework up-and-running. We will recommend good practices on application structure, development, testing, static code analysis, and deployment. Also, you will find sample implementations of the most commonly-used Grommet components.

Currently, Grommet is in early stages of development. As of this moment, we are neither deployed in Npm nor GitHub. This means that you will need to manually build Grommet in your local environment.

Building Grommet

To build Grommet is a prerequisite to install Git and Node.js. Then run the following commands to publish Grommet in your local npm: 


        git clone git@ligo.usa.hp.com:/opt/git/grommet.git
        

        cd grommet
        

        npm install
        

        npm link

You can now start a new project and link that to Grommet:


        mkdir sample-app
        

        cd sample-app
        

        npm link grommet

A node_modules folder will be created which contains the Grommet project from you local environment. Now it is time to create a package.json file that includes Grommet dependencies:

{'{\n  \"name\": \"sample-app\",\n  \"version\": \"0.1.0\",\n  \"main\": \"index.js\",\n  \"description\": \"Sample Grommet App\",\n  \"author\": \"Your Name\",\n  \"dependencies\": {\n    \"react\": \"^0.13.1\",\n    \"react-router\": \"^0.13.2\"\n  }\n}'}

Save this file and install the new project:


        npm install
