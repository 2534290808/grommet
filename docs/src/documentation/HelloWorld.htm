Hello World

The easiest way to learn a new framework is by writing a simple application on top of it. In this section you will have quick access to Grommet bits which will allow you to play with it in your browser. At the end of this exercise you will have created a Todo Application using Grommet components. We'll use prebuilt JavaScript files in this exercise which is not meant to be used in production. Please refer to Tutorial for building production-ready apps.

Create a new HTML file and paste the following code:


<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Hello World!</title>
  <link href='http://fonts.googleapis.com/css?family=Open+Sans:400italic,400,700' rel='stylesheet' type='text/css'>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/react/0.13.1/react.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/react/0.13.1/JSXTransformer.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/react-router/0.13.2/ReactRouter.min.js"></script>
  <script src="http://ligo.usa.hp.com/assets/grommet.min.js"></script>
</head>
<body>
  <div id="content"></div>
  <script type="text/jsx">
    var App = React.createClass({'{'}
      render: function() {'{'}
        return (
          <Grommet.App>
            <Grommet.Header centerColumn={'{'}true{'}'} primary={'{'}true{'}'}>
              <Grommet.Title>
                {'{'}"Your App Name"{'}'}
              </Grommet.Title>
              <div />
            </Grommet.Header>
            <ReactRouter.RouteHandler />
          </Grommet.App>
        );
      {'}'}
    {'}'});

    var TodoDashboard = React.createClass({'{'}
      render: function () {'{'}
        return (
          <Grommet.Section centerColumn={'{'}true{'}'}>
            <Grommet.Panel title="Hello World!">
              <p>If you&#39;ve made here, congratulations, you&#39;re inside Grommet world.</p>
              <p>Now, come back to <a href="http://ligo.usa.hp.com/docs/hpe/#/documentation">Hello World</a> page to continue your Grommet exploration.</p>
            </Grommet.Panel>
          </Grommet.Section>
        );
      {'}'}
    {'}'}); 

    var routes = (
      <ReactRouter.Route name="todoapp" path="/" handler={'{'}App{'}'}>
        <ReactRouter.DefaultRoute name="dashboard" handler={'{'}TodoDashboard{'}'}/>
      </ReactRouter.Route>
    );

    ReactRouter.create({'{'}routes: routes{'}'}).run(function (Handler) {'{'}
      var factory = React.createFactory(Handler);
      var element = document.getElementById('content');
      React.render(factory(), element);
    {'}'});
  </script>
</body>
</html>

Understanding the Dependencies

Grommet is developed on top of ReactJS which provides great features in JavaScript for building user interfaces. We are also using a JavaScript syntax extension called JSX. To enable front-end routing paths we are using React Router.

We expect that you have at least a basic understading on these technologies to be able to master Grommet, in addition to JavaScript of course. In terms of styling (css), Grommet already provides everything you need to quickly create great applications based on our application Style Guide. As a result, we don't expect you to write a lot of CSS code. After all, why reinventing the wheel?
