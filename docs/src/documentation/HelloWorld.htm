Hello World

The easiest way to learn a new framework is by writing a simple application on top of it. In this section you will have quick access to Ligo bits which will allow you to play with it in your browser. At the end of this exercise you will have created a Todo Application using Ligo components.

This Todo application will provide the following features:

  • A dashboard with the list of existing tasks
  • Login and Logout operations
  • Adding, removing, editing, and completing tasks
  • Managing labels and assigning them to tasks
  • Searching for existing tasks and/or labels
  • Filtering tasks by status (completed, in progress, created) and/or labels

Understanding the Dependencies

Ligo is developed on top of ReactJS created by Facebook which provides great features in JavaScript for building user interfaces. We are also using a JavaScript syntax extension called JSX. To allow front-end routing paths we are using React Router from Rackt.

We expect that you have at least a basic understading on these technologies to be able to master Ligo, in addition to JavaScript of course. In terms of styling (css), Ligo already provides everything you need to quickly create great applications based on our application Style Guide. As a result, we don't expect you to write a lot of CSS code. After all, why reinventing the wheel?

Initial Structure

In order to quickly get you started on Ligo, we'll use prebuilt JavaScript files in this exercise.

IMPORTANT NOTE: These files are not meant to be used in production. Please refer to Get Started for building production-ready apps.

That said, we should be ready to start developing our first Ligo application. Create a new HTML file with the following content:
